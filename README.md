# Outcome-Based Governance Framework (Draft)

This document proposes a governance framework where:
- Citizens decide collective goals,
- Real-world outcomes determine policy paths,
- Power and capital are subject to ex-post accountability.

Full text below is written in Chinese.

# 一种以人民终裁与现实结果为唯一合法性的社会制度设计
  ——基于二次方投票、长期预测市场与可清算权力的结果主义政体
## 一、写在最前：本理论的立场声明

本理论不试图证明某种意识形态的道德优越性，也不试图回答“什么是善”。
它只回答一个问题：
人类社会是否可能构建一种制度，使错误决策无法长期占优，而正确判断必然获得持续优势？

制度框架的目标，是将集体目标、路径选择、执行机制与结果清算重新绑定在同一制度闭环中。


本理论只承认一种正当性来源：
**现实结果在全体人民真实生活中的持续体现。**

凡是能够持续改善人民真实生活的结构，无论其形式为何，均被视为正当；
凡是持续恶化人民真实生活、却无法被纠正的结构，才被视为不可接受。

## 二、对现有一切政体的根本批判：反馈断裂

现有民主制、威权制、资本主义或其混合形态，在形式上彼此对立，但在结构性失败上高度一致：

决策者与决策后果脱钩
**错误决策往往由集体或制度承担，个人无需承担对应代价。**

权力与资本难以被追溯性清算
**即便造成系统性损害，既得利益者仍可通过合法形式保留收益。**

民意被廉价表达与情绪动员劫持
多数人的即时情绪，压倒长期真实利益。

这些问题不是因为人性不够高尚，而是因为**制度没有把现实结果作为唯一裁决者。**

## 三、制度总结构：四层不可互换的职能分离

本理论提出一种严格分层、不可越权的制度结构。
任何一层都不具备完整主权，只有整体形成闭环。

**第一层：** 目标确立（人民终裁）

人民只负责“想要什么样的生活”，而不负责“如何实现”。
人民通过高成本、强约束的集体表达机制（如二次方投票），决定社会的方向性目标与优先级，例如：

环境质量是否优先于增长速度
医疗与教育的最低保障标准
对安全、风险与幸福的偏好取向

二次方投票的目的不是“民主表态”，而是过滤廉价意见。
只有真正愿意为某一目标付出代价的偏好，才能形成社会方向。

这一层不讨论技术、路径、手段、模型、指标细节。
它只给出一句话形式的授权：

**“我们要这样生活。”**

**第二层：** 路径选择（预测市场）

谁认为自己知道“怎么做”，就必须为此承担现实风险。

所有关于如何实现第一层目标的具体方案，进入长期预测市场：

不同政策方案通过长期预测市场进行竞争
判断者需以自有资源为预测承担风险
正确判断获得收益，错误判断仅承担有限损失
市场周期与社会发展节奏匹配，允阶段性分红禁止短期抛售

**第三层：** 执行与拆解（AI 作为工具）
AI 的角色被严格限制为：

将抽象目标拆解为可执行指标
自动化处理高频、低风险、可替代事务
拦截明显违反生存底线或物理逻辑的方案

AI 仅承担指标拆解、执行协调与底线筛选
不具备价值判断权、目标设定权或暴力执行权
人类保留对执行结果的复核与纠错通道


**第四层：** 结果清算（因果追溯）
当政策被证明持续损害整体结果，且支持者从中获得显著不对称收益时，触发清算
清算的目的在于修复损失，而非惩罚本身

因此，资本不是必须被消灭的对象，而是被阶段性验证的生产工具。
一旦历史阶段证明资本存续本身降低繁荣度，其存在本身即可被否定。

## 四、关于指标
指标不是神圣的，现实感知才是

任何指标调整都必须服务于**人民授权**的目标。
指标无法长期掩盖现实生活的恶化：

人不是靠指标生活

而是靠空气、健康、环境、安全与尊严生活
一旦人民真实感知恶化，废除或修正路径的提案将不可避免地出现。

**核心指标：繁荣度**

框架采用“繁荣度”作为结果验证的核心指标，而非单一经济数据。
该指标为多维复合指标，涵盖：

民生与福利
经济与就业
科学与技术进步
风险防御能力
生态与长期可持续性

指标权重遵循边际递减原则，防止极端化发展路径。

## 五、不可被优化的人类边界

即便在最彻底的结果主义体系中，也必须存在一条不可被跨越的底线：

制度不得证明“某些人不应存在”。

因此：

不参与判断
不追求效率
不贡献于文明目标

都不得构成被贬值、被排除、被消除的理由。

这一边界不以慈善或情感为依据，而是为了防止制度在逻辑上走向自我合理化的极端。

## 六、自我定位与开放态度

本理论不是终极答案，而是一个制度原型：
繁荣度的测量存在误差，需要持续优化
市场判断可能阶段性失效

该框架被视为可证伪、可修改、可被替代的制度原型。
其正当性只来自持续改善现实结果的能力。

## 结语

本文并不承诺一个完美社会，而尝试避免一个**因制度性错误无法被纠正而走向自我损耗的社会。**
