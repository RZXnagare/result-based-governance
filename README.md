# Outcome-Based Governance Framework (Draft)

This document proposes a governance framework where:
- Citizens decide collective goals,
- Real-world outcomes determine policy paths,
- Power and capital are subject to ex-post accountability.

Full text below is written in Chinese.

# 一种以人民终裁与现实结果为唯一合法性的社会制度设计
  ——基于二次方投票、长期预测市场与可清算权力的结果主义政体
## 一、写在最前：本理论的立场声明

本理论不试图证明某种意识形态的道德优越性，也不试图回答“什么是善”。
它只回答一个问题：

人类社会是否可能构建一种制度，使错误决策无法长期占优，而正确判断必然获得持续优势？

**本理论拒绝以下前提：**

权力天然邪恶

垄断天然不正当

资本天然不可触碰

程序正义高于现实结果

本理论只承认一种正当性来源：

现实结果在全体人民真实生活中的持续体现。

**凡是能够持续改善人民真实生活的结构，无论其形式为何，均被视为正当；
凡是持续恶化人民真实生活、却无法被纠正的结构，才被视为不可接受。**

## 二、对现有一切政体的根本批判：反馈断裂

现有民主制、威权制、资本主义或其混合形态，在形式上彼此对立，但在结构性失败上高度一致：

决策者与决策后果脱钩
**错误决策往往由集体或制度承担，个人无需承担对应代价。**

权力与资本难以被追溯性清算
**即便造成系统性损害，既得利益者仍可通过合法形式保留收益。**

民意被廉价表达与情绪动员劫持
多数人的即时情绪，压倒长期真实利益。

制度无法区分“善意的失败”与“成功的作恶”
作恶者只要结果合法，即可持续获利。

这些问题不是因为人性不够高尚，而是因为制度没有把现实结果作为唯一裁决者。

## 三、制度总结构：四层不可互换的职能分离

本理论提出一种严格分层、不可越权的制度结构。
任何一层都不具备完整主权，只有整体形成闭环。

第一层：**人民终裁层** —— 目标的唯一合法来源
核心原则

人民只负责“想要什么样的生活”，而不负责“如何实现”。

人民通过高成本、强约束的集体表达机制（如二次方投票），决定社会的方向性目标与优先级，例如：

环境质量是否优先于增长速度

医疗与教育的最低保障标准

对安全、风险与幸福的偏好取向

二次方投票的目的不是“民主表态”，而是过滤廉价意见。
只有真正愿意为某一目标付出代价的偏好，才能形成社会方向。

这一层不讨论技术、路径、手段、模型、指标细节。
它只给出一句话形式的授权：

**“我们要这样生活。”**

第二层：路径竞争层 —— 现实判断的市场化验证
核心原则

谁认为自己知道“怎么做”，就必须为此承担现实风险。

所有关于如何实现第一层目标的具体方案，进入长期预测市场：

判断不以身份、资格或道德为依据

只以对未来现实结果的押注为依据

市场周期是长期的（5年、10年、20年），且允许中途退出与阶段结算

正确判断者获得收益；
错误判断者仅承担其下注的有限代价，不存在“终身出局”。

关于“判断集团”的立场

本理论**不反对判断垄断**。

如果某一判断集团长期正确、持续改善社会现实，那么其主导地位本身即是被现实验证的最优解。

一旦其判断出现错误，任何人都可以以极小成本反向下注并获得高额回报，从而完成纠错。

**判断垄断不是问题，不可纠正的错误才是问题。**

第三层：执行与约束层 —— AI 作为工具，而非权力主体
AI 的角色被严格限制为：

将抽象目标拆解为可执行指标

自动化处理高频、低风险、可替代事务

拦截明显违反生存底线或物理逻辑的方案

### AI 不具备以下权力：

不拥有目标决定权

不拥有价值判断权

不拥有暴力执行权

AI 是工具、是执行放大器、是守门员，但**永远不是统治者**。

第四层：因果清算层 —— 资本与权力的最终约束
核心原则

私人财产与既得权力不具备先验神圣性。

凡是通过以下路径获得的收益：

政策被证明有损社会整体

支持者为直接受益者

所得收益显著大于其承担风险

即便过程合法，也将被追溯性清算，用于修复社会损失。

本制度区分两种失败：

善意但错误的判断：只承担下注代价

成功但有害的获利：收益必须被回收

因此，资本不是必须被消灭的对象，而是被阶段性验证的生产工具。
一旦历史阶段证明资本不再有利于整体发展，其存在本身即可被否定。

## 四、关于指标、控制与所谓“阴谋”的明确立场
指标不是神圣的，现实感知才是

任何指标调整都必须服务于**人民授权**的目标。
指标无法长期掩盖现实生活的恶化：

人不是靠指标生活

而是靠空气、健康、环境、安全与尊严生活

一旦人民真实感知恶化，废除或修正路径的提案将不可避免地出现。

关于“若完全控制指标怎么办？”

### 若某一结构：

能长期控制指标

未被任何人察觉

且现实生活持续改善

那么该结构即被视为有效治理，而非作恶。

本理论不预设权力原罪，只承认结果裁决。

## 五、不可被优化的人类边界

即便在最彻底的结果主义体系中，也必须存在一条不可被跨越的底线：

制度不得证明“某些人不应存在”。

因此：

不参与判断

不追求效率

不贡献于文明目标

都不得构成被贬值、被排除、被消除的理由。

这一边界不以慈善或情感为依据，而是为了防止制度在逻辑上走向自我合理化的极端。

## 六、自我定位与开放态度

本理论不是终极答案，而是一个制度原型：

它可以失败

可以被证伪

可以被拆解

可以被历史淘汰

若失败，应被现实否定；
若有效，应被现实采用，而非被信仰。

## 结语

本理论并不试图建立一个“温情的社会”，
而是试图避免一个因错误决策而自毁的文明。

它相信的不是道德，而是现实；
不是程序，而是结果；
不是权力，而是可纠错性。

若人类终将失败，那么失败不应来自制度拒绝承认错误。
